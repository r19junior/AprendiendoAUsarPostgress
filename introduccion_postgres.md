# Aprendiendo PostgreSQL

PostgreSQL es una base de datos **Relacional Object-Relational (ORDBMS)** muy potente.

## Conceptos Clave

1.  **Base de Datos (Database)**: Un contenedor para todos tus objetos (tablas, funciones, etc.).
2.  **Esquema (Schema)**: Un espacio de nombres dentro de la base de datos (por defecto es `public`).
3.  **Tabla (Table)**: Donde se guardan los datos, con filas y columnas fijas.
4.  **SQL (Structured Query Language)**: El lenguaje estándar para interactuar con la base de datos.
5.  **psql**: Herramienta de línea de comandos para ejecutar consultas.
6.  **pgAdmin**: Herramienta gráfica (GUI) recomendada para principiantes.

## Primeros Pasos Sugeridos
1. Instalar PostgreSQL desde el sitio oficial.
2. Definir una contraseña segura para el usuario `postgres`.
3. Crear tu primera base de datos usando `setup.sql`.
